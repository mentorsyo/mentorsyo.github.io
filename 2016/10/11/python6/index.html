python基本学习(六)--进程和多线程

发表于 2016-10-11   |   分类于 知识

进程和多线程在相关的木马传播或者进程注入，自动开启进程都有很多的作用，线程是并发性行动，线程依赖进程多进程是很多不同的个体进程而线程是在进程里面多个线程互影响进行操作

进程

用进程就需要第三方模块 不得不说python的模块简直强大无比

from multiprocessing import Process,Pool
import os,time,random

process是进程的模块 pool是进程池 os就是输入输入的模块 time 时间模块 random 随机数模块

def run_process(name):
    os.getpid()可以获取到进程的id号
    print('子进程的名字%s 进程的id是%s' %(name, os.getpid()))
if __name__ == '__main__':
    print('当前的进程是%s' %os.getpid())
    #Process创建一个进程
    # target 需要填写一个函数的名字
    #args 填写targets相关的函数名的函数的参数
    #target里面函数的内容就是这个进程要做的事
    for x in range(0,5):
        p = Process(target = run_process,args = ('test',))
        print('进程将要开始')
        #创建进程之后要调用
        p.start()
        p.join()#进程结束之后才会执行下面的内容
        print('进程将要结束')

os.getpid()获取当前进程的进程号
if __name__ == '__main__'这个判断函数代表的就是如果当前进程为主进程的话执行下面的语句
进程执行也需要下面的几个步骤
1.依赖与函数先定义一个函数来代表进程要做的事情
2.创建一个进程
3.让进程开始执行你的定义函数
4.进程结束（自动结束）
在这个过程中有一个函数叫做 p.join()
代表的是进程自己结束以后才会执行我们下面的代码

进程池
因为我们在创建进程的过程中会如果想创建多个进程不能让他们同时进行
我们引入进程池是为了把进程装到一个池子里面让进程在里面

def run_process():
    print('当前的进程是%s' %os.getpid())
    #random。random（）随机产生一个0，1的数字
    # time.sleep 当前程序休眠
    start = time.time()
    time.sleep(random.random() % 2)
    end = time.time()
    print('当前进程要结束%s,进程运行了 %2f 秒' %(os.getpid(),(end-start)))
if __name__ == '__main__':
    print('当前的进程是%s' %os.getpid())
    创建一个进程池 函数中的参数代表的是同时执行的进程数目
    p = Pool(4)
    for x in range(4):
        向进程池里面添加进程
        p.apply_async(run_process,args = ())
让进程池结束添加进程，这个时候再调用apply_async函数不起作用
    p.close()
    会让进程结束以后执行下面的代码
    p.join()

和进程的开始和结束是一样的

线程

线程的模块是threading

import threading,time
实现多线程
线程也是需要通过函数来进行的
创建一个线程函数
def loop():
    current_thread()可以获取到当前的线程
    name表示当前线程名字
    print('当前是%s线程' %threading.current_thread().name)
    time.sleep(1)
    print('当前线程结束%s' %threading.current_thread().name)
loop()
开辟一个子线程 target表示子线程做的事情
thread = threading.Thread(target = loop, args = ())
thread.start()
thread.join()

写法基本和进程是一样的开始和结束中间填写进程要做的事情
对于多线程和多进程这两个概念其实无非来说就是
在进程里面定义多个线程来做同一件事等同于开多个进程来做同一件事情
