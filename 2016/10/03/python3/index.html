python中的类和对象实例

类

类的定义和相关语法的实现类的关键字就是class 然后书写类名 书写的方式类似与定义函数

class student():
    pass
a = student()

这就是实现了基本的类的定义和类的实例化
在类的括号里面需要填入的是所继承的父类
在定义类的过程中的会接触到一个函数叫做初始化函数 __init__这个函数
如果在类里面定义了这个函数要注意的是在对类实例化的过程中不能在用空的定义了必须传给参数

class student():
    def __init__(self,name,age):
        self.name = name
        self.age = age

对于类的函数来说每一次定义方法都必须在参数里面的第一个参数写上self
下面介绍一下类似php中封装的知识
在面向对象的语言中有三大特性
1.封装
2.继承

封装

下面我们对属性进行封装不让外界直接对我类里面的内容进行修改和访问
就需要在我们定义的属性前面加上两个下划线

self.__name = name

这样就不能在类外进行访问和修改 如果我们因为某些特殊需求必须实现对私有属性进行修改或者访问
1.把它改为公有属性
2.提供可访问的接口对它们进行操作也就是get set两种方法
提供两种访问的方法对其进行访问

def get_name(self):
    return self.__name
def set_name(self,name):
    self.__name = name

下面是一段比较完整的代码写法

class Student():
    #创建累的初始化方法 一旦写入这个方法再调用的时候就必须传入参数
    #第一个参数永远是self 但在赋值的时候不用管会自动把实例传递进来
    def __init__(self,name,age): #写了这个方法就必须调用这个方法 __init__初始化的方法
    #创建属性并且给属性赋值
    #再属性的前面添加一个下划线 表示 我虽然是公有属性 但是请把我当做一个私有属性看待 仅仅是一个约定
        self.__name = name# 双下划线代表私有属性 只能在类里面进行调用不能在外面进行调用 没有则是公共属性 如果要在外界进行访问的话需要调用get和set的方法
        self.__age = age
        self.__city = 'beijing'
        #在类里面创建函数第一个参数永远是self 但是不需要赋值 会自动把对象传给self
    def print_STU(self):
        print('name: %s age: %s' %(self.__name,self.__age))
    def get_city(self):
        return self.__city
    def get_name(self):
        return self.__name
    def set_age(self,age):
        if abs(age) 100:
            self.__age = 100
        else:
            self.__age = abs(age)
    def get_age(self):
        return self.__age
a = Student('bob',45)
a.print_STU()
print(a.get_city())
print(a.get_name())
a.set_age(1000)
print(a.get_age())

继承

类之间的继承 没有父类的类叫做基类 类的继承可以无限继承也就是子类可以当作父类来进行继承
类和类之间的继承只需要在类的括号里面填写上父类
子类可以直接调用父类的方法也可以在子类里面重写父类的方法或者写父类没有的方法如果重构方法就会使用当前子类的方法

class Animal(object):# 基类所有的子类都继承父类 子类可以继承父类的方法
    def eat(self):
        print('Animal eating')
class Dog(Animal):
    def eat(self):
        print('Dog eat meat')
class zangao(Dog):
    pass
class Horse(Animal):
    def eat(self):
        print('horse eat grass')
a = Animal()
b = Dog()
c = Horse()
d = zangao()
a.eat()
b.eat()
c.eat()
d.eat()

多态就是父类的多种形态的展现 因为子类都继承与父类，父类可以通过子类实现父类本身的方法和属性而一个父类又可以有很多的子类，并且多个子类可以以不同的形式来展现父类的方法 关于多态 一句话概括 父类指针指向子类对象

def foo(animal):
    if isinstance(animal,Animal):
