python基本学习(四)--文件操作

发表于 2016-10-05   |   分类于 知识

python学习第四课，今天笔记记录python的文件操作

文件操作

python中不能直接对文件进行操作，所以要系统提供一个文件的接口给python，python打开文件实例对其进行操作
我们现在桌面建造一个文件取名字为text.txt
下面我们要用到f.open()函数来进行文件操作

如果读取的文件和python所在同一个目录就可以直接进行读取 如果不在这一个目录则需要路径来进行读取 注意转义字符

f = open('C:\\Users\\Administrator\\Desktop\\text.txt','r') # 要对反斜杠进行转义

这时候我们已经获得了文件的实例，我们就可以对文件进行操作

通过read函数对文件进行读取操作
print(f.read())
按照字符的多少来进行读取
print(f.read(10))
读取一个文件的内容并且返回一个列表
将所有的读取出来的数据转化成一个列表 相当于按行读取
print(f.readlines())
每次操作完对象以后记得要关闭对象
f.close()

要记得是就是记得在最后文件完成以后要记得关闭文件对象的接口
每一次用写入的方式进行操作会导致文件清空

f = open('C:\\Users\\Administrator\\Desktop\\text.txt','r')
a = f.read()
f.close
f = open('C:\\Users\\Administrator\\Desktop\\text.txt','w')
f.write('%s hello new world ' %a)
f.close()

try finally: 无论try是否为真都会执行finall
这种方法为了防止文件操作的过程中出错的话导致文件不能关闭的错

try:    
    f = open('C:\\Users\\Administrator\\Desktop\\text.txt','r')
finally:
    if f :
        f.close

还有一种更加省事的方法就是用with as 的方法这种方法的好处就在于不需要输入关闭函数系统会根据自身情况来自己进行关闭

with open('C:\\Users\\Administrator\\Desktop\\text.txt','rb') as f:#读二进制文件要在r后面加上b
    print(f.read())

下面我们要掌握的就是对文件夹和文件进行增删改的操作
这需要调用模块 os 来对文件进行执行操作

import os
print(os.path.abspath('C:\\Users\\Administrator\\Desktop\\text.txt'))#查找文件的绝对路径
print(os.path.abspath('.'))

os.path.abspath()这个函数就是可以查看文件的绝对路径
当文件夹存在是无法创建该文件夹 mkdir

os.mkdir('C:\\Users\\Administrator\\Desktop\\newdir')

删除一个目录 rmdir 只能删除空文件夹

os.rmdir('C:\\Users\\Administrator\\Desktop\\newdir')

两个对文件夹进行操作的方式一种是创建 一种是删除
下面介绍一下拼接路径的方法在php的里面拼接路径的方法是用到.现在我们要用+来进行拼接

a = 'C:\\Users\\Administrator\\Desktop\\'+'a.txt'

下面对文件进行重命名remname 一共两个参数第一个参数代表的是你需要重命名的文件的路径后面代表的是你重命名以后文件的路径

os.raname(,)

删除文件的操作 只有一个参数代表的是你想要删除文件的路径

os.ramove()

最重要的一个知识获取当前路径下目录和文件夹 listdir()

print([x for x in os.listdir('.') if not os.path.isdir(x)])#定义函数变量x 然后执行for循环来列出文件夹中的有文件及文件夹
print([x for x in os.listdir('.') if os.path.isfile(x) and os.path.splitext(x)[1]=='.py'])#os.path.isfile 是不是文件

os.path.isfile判断文件是不是文件类型的
os.path.dir来判断当前文件夹里面的文件是不是目录

下面一个函数是直接把文件查分成文件名字和后缀
os.path.splitext()分成了元祖 元祖的第一个元素是文件名 第二个元素为后缀名

下面是自己写的一些创建文件和删除非空文件夹的函数

def my_mkdir(path):
    import os
    os.mkdir(path)
    a = os.path.join(path,'1.txt')
    b = os.path.join(path,'b.txt')
    c = os.path.join(path,'a')
    os.mkdir(c)
    with open(a,'w') as f:
        f.write('aaaa')
    with open(b,'w') as f:
        f.write('bbbbb')
def my_rmdir(path):
    import os
    if os.path.isdir(path):
        if os.listdir(path):
            for x in os.listdir(path):
                a = os.path.join(path,x)
                if os.path.isdir(a):
                    my_rmdir(a)
                else:
                    os.remove(a)
            os.rmdir(path)
        else:
            print('文件夹为空')
            os.rmdir(path)
    else:
        print('您输入的路径不是一个文件夹')
my_mkdir('C:\\Users\\Administrator\\Desktop\\newdir')
my_rmdir('C:\\Users\\Administrator\\Desktop\\newdir')
