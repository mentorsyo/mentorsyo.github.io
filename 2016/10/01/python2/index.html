今天写python的函数

函数

python的函数写法和其他语言的也不一样定义函数的关键字在于 def 然后跟上函数的名字和函数参数以及函数体，返回值

def my_abs(a):
    if a 0:
        return a 
    else:
        return -a

这就是一个简单的函数实现 在定义函数的过程中要注意不要把函数名字定义的和系统函数一样 这样会覆盖掉系统函数的定义最后导致系统函数不能实现

函数的参数
函数的参数是分为可变参数，默认参数，关键字参数
这三种参数各自有各自的优劣
默认参数
在使用默认参数的过程中我们要注意默认参数不能把默认参数放在第一位这会导致赋值的过程中不知道赋值给谁所以再python中直接就不能把默认参数放在第一位

def my_functioon(sex,age=10):
    print(sex,age)

可变参数
可变参数的使用是在变量的前面加上*号这样就变成了可变参数 可变参数可以被一直使用 在函数中变成元组
可变参数是不能传入数组只能传入值来使用

def new_abc(*a):
    b= 0
    for x in a:
        b+=x
    return b
L = [11,22,4846,84,89,45463]
print (new_abc(*L))# 如果是一个列表还想用多参数传参的话就在列表前添加*这样就会让列表成功传参

关键字参数
关键字参数就是可以赋予他键值和值两个东西让其生成字典 定义方式 **参数名

def student(name,age,**kv):#在参数名字前面添加两个** 这样就表示这是一个关键字参数  可变参数代表的是一个字典连接
    print('name: %s age: %s kv: %s' %(name,age,kv))
student('Bob',18,gender='man',city='beijing')# 传值的过程要先写键值在写值 最后函数会自动生成一个字典也就是 键值不需要加引号
def fact(n):
    if n==1:
        return n
    return n * fact(n-1)
print(fact(10)

再调用函数的过程中我们要记住和其他语言一样都是直接进行调用

还有一种函数是递归函数就是连续调用本身的一种函数但最好不要书写 容易出现死循环
