python基本学习(五)--数据库

发表于 2016-10-07   |   分类于 知识

在python里面我们也需要用python对数据库来进行操作让数据库进行增删改查的相关操作

数据库操作

我们要引入一个可以用来操作mysql的相关模块

import pymysql

对数据库来进行操作其实和php的操作是一样的 都要几个步骤
1.连接数据的语句 得到一个可以操作数据库的句柄
2.对数据库建立链接
3.执行sql语句
4.对执行的sql语句进行保存
5.关闭句柄
这一套程序下来就是这样的

import pymysql
#设置连接那个数据库
coon = pymysql.connect(host = 'localhost',port = 3306,user = 'root', passwd = '123456',db='Classes',charset = 'UTF8')
#建立链接
connect = coon.cursor()
#执行sql语句
connect.execute('create table if not exists student (id varchar(20) primary key,name varchar(20))')
connect.execute('insert into student(id,name) values("1","your")')
connect.execute('update student set name ="my"')
coon.commit()
#connect.execute('delete from student where id=1')
connect.execute('select * from student')
values = connect.fetchall()
print(values)
coon.close()

以上就是对mysql进行操作的相关代码
下面介绍一下python自带的一个数据库叫做sqlite3
操作和mysql的操作基本一样
import sqlite3

# 建立数据库链接如果不存在则会自动创建
coon = sqlite3.connect('text.db')
#建立链接
connect = coon.cursor()
connect.execute('create table if not exists student (id varchar(20) primary key,name varchar(20))')
connect.execute('insert into student(id,name) values(\'1\',\'your\')')
connect.execute('select * from student where id = ?',('1',))
values = connect.fetchall()
print(values)
connect.close()
coon.commit()
coon.close()
