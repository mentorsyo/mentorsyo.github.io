python基本学习(七)--套接字

Posted on 2016-10-15   |   In 知识

socket套接字是一种协议，它可以让通信长链接，现在的IM一般都在用这个来进行协议通信 也就是即时通信的产品
下面就用scoket的协议模拟了一个客户端和服务端的操作

服务端

import socket,threading
#通过socker模块创建出来的socket对象
#AF_INET 表示的是ipv4的协议
#SOCK_STREAM表示使用tcp

#执行到socket_link这个函数的时候表示已经有一个人连接进来
def socket_link(sock,addr):
    #发送消息
    #addr 是一个元组
    print('%s : %s 链接进来了' %addr)
    sock.send(b'welcome')
    while True:
        #接受消息 设置大小 单位字节
        message = sock.recv(1024)
        print(message.decode('utf-8'))
        if not message or message.decode('utf-8') == 'exit':
            sock.close()
            print('%s : %s 已经断开链接' %addr)
            break
        #关闭socket连接

s = socket.socket(socket.AF_INET,socket.SOCK_STREAM)
s.bind(('127.0.0.1',6661))
s.listen(5)
print('等待链接')
while True:
    #accept可以获取到连接对象和连接的地址
    sock,addr = s.accept()
    thread = threading.Thread(target = socket_link,args = (sock,addr))
    thread.start()

客户端

import socket
#socket的客户端
#创建一个socket的连接请求

s = socket.socket(socket.AF_INET,socket.SOCK_STREAM)
#连接请求 这时候已经连接好了可以接受消息了
s.connect(('127.0.0.1',6661))
print(s.recv(1024).decode('utf-8'))

while True:
    message = input('enter your message:')
    if message == 'exit':
        s.send(message.encode())
        break
    else:
        s.send(message.encode())

s.close()

这就相当于一种通讯 服务端保持不断开知道客户端发送断开的命令，最后两边断开
