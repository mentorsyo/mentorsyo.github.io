msf的一些小坑

Posted on 2016-03-21   |   In 知识

今天安装了kali的2.0版本的msf使用
首先更新一下软件源
因为kali2.0以后以前的软件源都不行了 所以用下面的

deb http://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free

vi /etc/apt/source.list

在里面添加软件源

然后执行更新源命令

apt-get update

安装图形化服务管理页面

apt-get install sysv-rc-conf
apt-get install htop

命令是为了安装htop界面

启动sysv-rc-conf

sysv-rc-conf

可以看到当前所有服务的图形化界面

然后我们启动服务

service postgresql start

关键在这里要首先初始化数据库才可以连上

msfdb init
service metaspolit start

或者

msfconsole

启动以后如果出现了慢查询就是cache出了问题
先查询看看数据库是否连接上

db_status

如果连接成功则用

db_rebuild_cache

退出程序
重启postgresql服务

service postgresql restart

应该就可以解决了
手动连接数据库的方法就是

db_connect 用户名:密码@localhost/数据库名字
