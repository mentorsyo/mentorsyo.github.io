简介Python

python是一门脚本语言，它是由著名的龟叔，在1989年为了打发圣诞节的无聊时光写出来的，在这几年里面python的使用增长速度日益剧增


可以看出python的地步
在编程语言里面分为四种
1.机器语言
2 汇编语言
3 脚本语言
4 高级语言
其中机器语言就是01组成的机器码是给机器转成电路语言的进行物理传输的
汇编语言是可以直接编译成机器语言的
高级语言是现在很多运用在编程的语言比如说JAVA C C++ C# Object_C 这类语言全部都是高级语言
脚本语言是需要依赖于一门高级语言为基础 不需要进行编译直接解释执行 因此脚本语言执行速度比较慢
高级语言是直接编译成机器语言因此速度快
脚本语言有 python php js perl 这类语言
在所有的脚本语言里面python的地位是排名第一因此python的重要性可见一斑

变量和输入输出

因为python是一门弱语言，所以在python当中我们定义变量的时候不需要把数据类型加上只需要把变量名称赋值即可
例如

a = 100

这句话解释出来的意思是把变量a指向内存中100所在的内存地址
接下来就是一门语言中必须的输入和输出
在python3.5中输入和输出都被定义成了函数所以在输出和输入的
输出就成了

print()

输入就成了

input()

在输出字符或者变量的时候我们可以有好几种输入的方法
例如

print('hello world')
a = 'hello world'
print(a)

还有一种格式化输出

print('i say:%s' %'hello world')

输出函数就这样了
输入函数中也可以写入我们想要说的话
例如

input('请输入你的姓名')

输入输出也就这样了
在python中变量和输入输出是基础

数组

在python中数组分为三类
1.列表
2.元组
3.字典
这三种数组代表了三种不同的存储数据的容器
先简单的分以下
列表就是最简单的数组在相当于php中的索引数组 都是利用有序数字进行下标的然后取值的过程中也是利用数字下标来取值
列表是有先后顺序的 它的定义方式是

a = [1,2,3,4,5,6]

元组是也是一种列表但是和列表的本质区别在于 元组内的元素一旦定义就不可能在发生改变不能进行 增删改这三个操作
定义的方式

a = (1,2,3,4,5,6)

字典不同与另外两种方式 他相当于php中的关联数组 它存在key值和值 也就是字典需要的是两个空间来存值 一个空间存入索引值另一个存入值
定义的方式

a = {'name':'Bob','age':15,'sex':'man'}

这三种定义的方式就是这样
下面讲一下在数组中会用到的一些函数
append()函数
增添数组在数组的末尾相当于追加 括号里面写的是你想要增添的数据

a.append(6)

insert()函数
插入函数在你想要插入的位置插入你需要插入的数据需要两个参数

a.insert(3,6)

pop()删除函数
删除函数有两种 一种不添加参数 直接删除数组最后一个
另外一种方式是写入你想要删除的位置删除所需的值

a.pop(3)

len()函数 代表数组的长度或者字符串的长度
get() in()这两个函数是用来判断数据是否在这个数组中

print('刘能' in scores)#返回的是ture或者false
print(scores.get('刘能'))#返回的值是none

在写数组的过程中需要注意的小地方
在定义元组的过程中如果只定义一个数值的话需要在数值后面加上一个,

a = (1,)

定义字典的过程中
不可以通过不存在的key取value 但是可以通过不存的key赋值相当于创建了一个新的key值插入到字典中

分支结构和循环结构

分支结构就是if判断语句
在这个语言中if判断的写法

if 1 < 2:
    print('1小于2')
else:
    print('不小于')

在其他语言中存在elseif Python中也不例外只是elseif写成了
elif
循环结构
分为两种for循环和while循环

for x in range(1,10):
    print(x)

while(x<10):
    x++
    print(x)

用一句话来说for循环和while循环的区别
土豪吃包子 吃饱才不吃 其他人吃包子只有那么多钱只能吃这么点 for循环就是代表的是明确的知道循环的次数是多少 while是不知道限定条件到达田间就结束循环
